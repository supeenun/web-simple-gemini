# Portfolio Website Deployment Plan

## Objective
Create and deploy a professional, dark-themed portfolio website for a Senior System Analyst on GitHub Pages.

## Tech Stack
- HTML5
- CSS3 (Custom properties for modern dark theme, Google Fonts for JetBrains Mono)
- Vanilla JavaScript (for interactivity)

## Implementation Steps

### 1. Project Setup
- Create a new directory for the portfolio.
- Initialize git repository.

### 2. Development Phase
- **Structure (`index.html`):**
  - `<header>`: Navigation.
  - `<section id="hero">`: "Supeenun King - Senior System Analyst".
  - `<section id="projects">`: Grid container for 3 projects (Title, Description, Links).
  - `<section id="contact">`: Email, LinkedIn, GitHub links.
- **Styling (`style.css`):**
  - Implement dark color palette (slate/charcoal background, soft text colors).
  - Apply `JetBrains Mono` font via Google Fonts.
  - Mobile-responsive layout (flexbox/grid).
- **Interactivity (`script.js`):**
  - Add basic hover effects or smooth scrolling if needed.

### 3. Deployment
- Push local files to a new GitHub repository.
- Enable GitHub Pages in repository settings.

### 4. Verification
- Verify site live status at `https://<username>.github.io/<repo-name>/`.
- Test responsive behavior on mobile/desktop.
